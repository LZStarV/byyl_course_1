# 注释规范与模板

## 文件头注释模板
/*
 * 版权信息：Copyright (c) 2023 林展星
 * 文件名称：[文件名]
 * 功能描述：[详细说明本文件/模块的主要功能]
 *
 * 当前版本：1.0.0
 * 作    者：林展星
 * 完成日期：2023年12月7日
 *
 * 版本历史：
 * 1.0.0 2023-12-07 林展星 初始版本
 * [此处预留版本历史填写位置]
 */

## 函数头注释模板
/**
 * 功能描述：[详细功能说明]
 * @param {类型} 参数名 - 参数说明
 * @return {类型} 返回值说明
 * @throws [异常类型或条件]
 * @note 调用注意事项
 */

## 宏与变量
- 宏：在 `#define` 上方说明用途、取值范围、单位、示例；若作为开关，注明默认值与覆盖方式。
- 变量：在全局/静态变量前注明作用域、用途、生命周期、线程安全约束。

## 关键算法
- 对复杂流程（如 Thompson/Subset/Hopcroft、LR 构造与闭包、分发表生成），在实现处添加行内注释解释每一步含义；覆盖不少于代码行数的 30%。

## 版本历史维护
- 每次修改在文件头“版本历史”段追加一行，格式：`[版本号] [日期] [修改者] [修改摘要]`。
- 示例：`1.0.1 2023-12-08 林展星 修复XX模块的内存泄漏问题`。
